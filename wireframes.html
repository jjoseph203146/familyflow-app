Stack note: PWA and React Native are different technologies. Since you want a PWA (installable via browser, no app store), the recommended stack is React + Vite + vite-plugin-pwa for the frontend and Supabase (auth, database, file storage) for the backend. React Native is for native app store submissions — not needed for a PWA. We can revisit this before we build.
